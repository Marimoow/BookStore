# BookStore

### This University Online Used Book Store is a student start-up at Central Univeristy of Finance and Economics.
### About the details, see my website.

### Project Structure
```
├── UsedbookApp.iml
├── app
│   ├── app.iml
│   ├── build
│   │   ├── generated
│   │   ├── intermediates
│   │   ├── outputs
│   │   └── tmp
│   ├── build.gradle
│   ├── proguard-rules.pro
│   └── src
│       ├── androidTest
│       ├── main
│       └── test
├── build.gradle
├── gradle
│   └── wrapper
│       ├── gradle-wrapper.jar
│       └── gradle-wrapper.properties
├── gradle.properties
├── gradlew
├── gradlew.bat
├── local.properties
└── settings.gradle

```
### Denpendencies and configurations

- Android Studio 3.5
- Online SQLite3 database on Bmob Cloud
- Simulator Pixel 2 API 24
